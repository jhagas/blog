MENGGUNAKAN COMMAND LINE ADALAH JALAN NINJAKU

Sebuah panduan singkat mengenal dunia command line yang tak terbatas

Minggu, 2 Mei 2021

Command Line Tidak Menakutkan

Kita mungkin sering berpikir, terminal emulator (selanjutnya akan disebut terminal) atau tampilan utama Command Line itu tampak seperti saat menghadapi komputer rusak, jelek dan mengintimidasi. Namun sekarang ubah mindset anda, command line itu tidak menakutkan, bukan pula untuk programmer atau hacker berjaket hitam dan mengenakan topeng anonymous.

Command line adalah cara yang retro untuk berinteraksi dengan komputer. Bahkan jika kita telah menguasai, kita bisa melakukan kegiatan kita dengan lebih cepat, contohnya navigasi file, mengetik, coding, dll. Salah ketik pun tidak akan membuat file-file dalam komputer anda lenyap, namun di sistem Unix (Linux, MacOS, Solaris, dll), kita cukup berhati-hati dengan kata sudo. Disini kita akan menjelajahi, darimana kita akan memulai perjalanan masuk ke dalam rabbit hole penggunaan Command line.

Kustomisasi Terminal

Tentu anda tidak ingin terminal anda nampak mengintimidasi seperti ini kan..

Nah kustomisasi terminal adalah hal yang penting. karena selain membuat tampilan terminal anda menjadi lebih menarik, pada proses kustomisasi anda juga akan belajar untuk sedikit mengenal apa itu terminal dan command line. Ada beberapa hal yang bisa dijadikan patokan untuk mengkustomisasi tampilan command line

Pilihan Terminal Emulator

Yang pertama adalah pilihan Terminal Emulator. Sekilas Terminal Emulator memiliki tujuan yang sama, yaitu berinteraksi dengan Shell yang disediakan oleh sistem. Shell adalah kumpulan dari beberapa perintah yang bisa memanggil dan menjalankan serta mengelola program yang telah dipasang di komputer. Meski terlihat sama, setiap terminal ternyata memiliki fitur yang berbeda-beda pula, bahkan dengan fitur yang sama persis, implementasi yang dijalankan dapat berbeda. Berikut adalah terminal yang saya sarankan untuk setiap Sistem Operasi berdasarkan fitur yang ada.

  • Untuk GNU/Linux, gunakan terminal Kitty
  • Untuk Mac OS, gunakan terminal iTerm2
  • Untuk Microsoft Windows, gunakan Windows Terminal.

Font dan Tema Warna

Meski terkesan sepele, font dan tema warna adalah hal yang dapat membuat kita nyaman berada di dalam terminal. Font dan tema warna mempengaruhi cara kita membaca dan menginterpretasikan teks input yang kita tulis dan teks output dari program yang kita jalankan.

Di sini saya menggunakan sistem Linux dengan terminal emulator Kitty. Font yang saya gunakan adalah JetBrainsMono Nerd Font. Sedangkan tema warna yang saya pasang adalah Nord.

Aplikasi Pada Terminal

Untuk produktivitas maksimal, tentu kita perlu menggunakan beberapa aplikasi TUI (Terminal User Interface) untuk menggantikan aplikasi GUI (Graphical User Interface). Berikut adalah aplikasi TUI yang saya pakai untuk kegiatan sehari-hari

  1. neovim (Text editor/IDE)
  2. nnn (File manager)
  3. Dijo (Activity logger)
  4. Tuxi (CLI assistant, Google scraper)
  5. KBBI-CLI (Makna kata KBBI di Terminal)
  6. Newsboat (RSS Feeder)

Tidak semua dari aplikasi di atas bisa dijalankan pada semua platform, namun aplikasi di atas bisa dipastikan berjalan dengan baik dan lancar di GNU/Linux.

Wrap-up

Karena peningkatan produktivitas yang signifikan saat menggunakan aplikasi Terminal daripada GUI, kita perlu mengerti cara menggunakan terminal itu sendiri. Langkah pertama adalah mengubah mindset di mana Terminal itu tidak menakutkan, dengan membuat tampilannya menjadi menarik. Kita juga perlu menggunakan aplikasi CLI/TUI untuk menambah produktivitas kita dengan power yang luar biasa dari Terminal.
